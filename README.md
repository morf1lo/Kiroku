# How to run?

you need [Rust](https://rust-lang.org) and [Node.js](https://nodejs.org) installed

```bash
git clone https://github.com/morf1lo/Kiroku.git
cd Kiroku
npm i
```
#### Development
```
npm run tauri dev
```
#### Build
```
npm run tauri build
```
it will appear in `src-tauri/target/release`
